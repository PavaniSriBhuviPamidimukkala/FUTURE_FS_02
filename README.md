# Bean & Brew Café Website

A modern and responsive café business website developed as part of the Future Interns Full Stack Web Development Internship – Task 2 (2026).

## Project Overview

This project was created for a local café business concept called **Bean & Brew Café**.
The goal of the website is to help the business establish a professional online presence, showcase its offerings, and attract more customers.

The website focuses on:

* Professional UI design
* Responsive layout
* Customer engagement
* Business branding
* Real-world business presentation

---

## Features

* Responsive modern design
* Sticky navigation bar
* Hero landing section
* About section
* Café menu showcase
* Image gallery
* Customer testimonials
* Contact information
* Google Maps integration
* WhatsApp contact button
* Mobile-friendly interface

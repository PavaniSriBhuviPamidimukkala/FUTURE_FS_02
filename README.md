## 📖 About This Project

This repository contains a fully responsive and modern fitness center website developed using pure HTML5, CSS3, and Vanilla JavaScript.

The project was designed to simulate a real-world gym business website that serves two different customer groups simultaneously:

- 💪 Everyday gym members looking for affordable memberships
- 👑 Premium clients seeking elite personal training and VIP services

The website focuses on:
- Modern UI/UX Design
- Mobile Responsiveness
- Conversion Rate Optimization (CRO)
- Interactive Frontend Functionality
- Real Business Website Structure
